# Mini Projects Collection 🚀

A collection of creative frontend mini-projects built using **HTML, CSS, and JavaScript**.

This repository contains multiple small projects, UI components, animations, games, buttons, forms, clocks, and interactive web experiments.  
The main goal of this repository is to practice frontend development, improve creativity, and build strong fundamentals in web design and JavaScript.
